# Tunex Barbing Salon

An ultra-premium, cinematic, luxury barber website for Tunex Barbing Salon — a modern grooming brand in Nigeria.

## Features

- **Cinematic Hero Section** — Full-screen parallax background with animated stats and trust badges
- **Smooth Scroll Animations** — Framer Motion powered scroll-triggered reveals
- **Glassmorphism Design** — Premium frosted glass cards with gold accents
- **Interactive Gallery** — Masonry grid with lightbox and category filters
- **Auto-scrolling Testimonials** — Luxury customer review carousel
- **WhatsApp Booking Integration** — Direct booking via WhatsApp with pre-filled messages
- **Custom Cursor** — Animated dual-layer cursor effect (desktop)
- **Floating Gradient Orbs** — Subtle ambient lighting effects
- **Scroll Progress Indicator** — Gold progress bar at the top
- **Loading Screen** — Branded loading animation with progress
- **Mobile Responsive** — Fully optimized for all devices
- **SEO Optimized** — Meta tags, semantic HTML, and performance optimized

## Tech Stack

- React 18 + Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons
- Google Fonts (Bebas Neue + Poppins)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
